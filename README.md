# setup

```bash
node -v
npm -v
mkdir vue-npm && cd vue-npm
npm init -y
npm install vue --save
npm install -g @vue/cli @vue/cli-service-global
vue create vuetify-test (requires manual actions)
cd vuetify-test
vue add vuetify
```

# execute 

```bash
npm run serve
```
